# Daily Habit Tracker V2 

## Overview
A web application that allows users to create and track daily habits with visual streak counters to build motivation and consistency. Users can check off completed habits each day and monitor their progress over time.

## Problem
People struggle to maintain consistency with daily habits due to lack of clear tracking and visual progress feedback

## Target User
Individuals looking to build positive daily routines and track personal habit consistency

## Features
- User registration and authentication
- Create/edit/delete personal habits
- Daily habit check-off functionality
- Streak counter display
- Personal dashboard showing all habits and progress
